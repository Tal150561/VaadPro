# VaadPro Bridge – הוראות התקנה

## דרישות
- Windows 10/11
- Node.js (הורד מ-https://nodejs.org ← גרסת LTS)

## התקנה (פעם אחת בלבד)

1. **צור קובץ config.json** בתיקייה זו עם הפרטים שקיבלת באימייל:
```json
{
  "cloudUrl": "https://...",
  "bridgeSecret": "...",
  "tenantId": "..."
}
```

2. **לחץ פעמיים על install.bat** — ממתין ~2 דקות

## הפעלה יומית

**לחץ פעמיים על start.bat**

- יפתח חלון שחור — **אל תסגור אותו!**
- פתח את VaadPro בדפדפן
- לחץ "חיבור ווטסאפ" וסרוק את ה-QR (פעם ראשונה בלבד)
